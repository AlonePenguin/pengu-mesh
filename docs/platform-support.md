# Platform Support

## Tier 1

- `darwin/arm64`
- full validation target
- benchmark reference platform
- first release quality bar

## Tier 2

- `linux/amd64`
- `linux/arm64`
- `windows/amd64`

Tier 2 targets remain structurally supported in the code layout, but the local
gate and current audit bundle are still centered on host-native `darwin/arm64`
verification.
